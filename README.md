# Gouri Biswas – Portfolio Website

A modern and responsive developer portfolio built using Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

This portfolio showcases:
- Projects
- Technical skills
- Experience
- Achievements
- Resume
- Contact information

It includes a clean UI, dark/light mode support, smooth animations, project links, and a working contact form using EmailJS.

---

## Tech Stack

- Next.js
- React.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- EmailJS

---

## Features

- Responsive design
- Modern UI/UX
- Dark mode support
- Live project demos
- GitHub project links
- Resume integration
- Contact form functionality
- Smooth scrolling and animations

---

## Use This Template

You may use this portfolio template and customize it according to your own projects, skills, experience, and resume.

---

## Clone the Repository

```bash
git clone <your-repository-link>
```

Navigate into the project folder:

```bash
cd <project-folder-name>
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
